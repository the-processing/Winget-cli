Aperçu du gestionnaire de packages Windows v0.1.41331

Il s'agit de la version d'aperçu initiale du Gestionnaire de packages Windows. Windows Package Manager est une solution complète de gestionnaire de packages qui se compose d'un outil de ligne de commande et d'un ensemble de services pour l'installation d'applications sur Windows 10.

Les développeurs utilisent l' outil de ligne de commande winget pour découvrir et installer un ensemble d'applications organisé. Une fois installé, les développeurs peuvent accéder à winget via le terminal Windows, PowerShell ou l'invite de commande.

Caractéristiques
winget expose les commandes et options suivantes pour l'aperçu.

install Installe l'application donnée
show Affiche des informations sur une application
source Gérer les sources d'applications
rechercher Trouver et afficher des informations de base sur les applications
hash Helper pour hacher les fichiers d'installation
validate Valide un fichier manifeste
--help Fournit une aide en ligne de commande
--info Fournit des données supplémentaires, utiles pour le dépannage
--version Fournit la version du client
Pour une documentation complète, pensez à https://docs.microsoft.com/windows/package-manager
